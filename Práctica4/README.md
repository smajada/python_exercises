# Ejercicio 2
Escribe un programa que pida dos números y escriba qué números entre ese par de números son pares y cuáles impares.

Ejemplo:

Escribe un número: 4

Escribe un número mayor que 4: 8

El número 4 es par

El número 5 es impar

El número 6 es par

El número 7 es impar

El número 8 es par
